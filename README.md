#  基于 FreeRTOS 的多通道智能数据采集与监控系统

##  项目简介
本项目是一个基于 STM32 微控制器与 FreeRTOS 实时操作系统的多通道数据采集与异常监控系统。项目充分利用了 RTOS 的多任务调度机制，实现了 ADC 数据的高效采集、软件滤波处理、实时串口打印以及异常状态报警功能。

该项目旨在解决传统裸机开发中数据采集与逻辑处理耦合度过高的问题，通过任务解耦和进程间通信，保证了系统在高频数据采集下的实时性与稳定性。

##  核心功能与技术亮点
- **多任务实时调度**：基于 FreeRTOS 构建，合理划分了数据采集、串口打印、报警监测等独立任务。
- **高效数据采集**：使用 **DMA + ADC** 配合多通道扫描，极大降低了 CPU 的占用率。
- **软件滤波算法**：在采集任务中加入了软件滤波处理，有效剔除异常噪点，保证数据精准度。
- **进程间通信 (IPC)**：
  - 使用 **消息队列 (Queue)** 实现采集任务与打印任务之间的高效数据传递。
  - 使用 **互斥锁 (Mutex)** 保护串口打印资源，防止多任务并发访问导致的数据乱码。
- **异常报警机制**：独立的高优先级报警任务，能够实时监测数据阈值并迅速响应。
- **内存安全监控**：实时监控 FreeRTOS 堆内存使用情况，有效防范内存泄漏与碎片化风险。

## ️ 硬件与开发环境
- **主控芯片**：STM32F103
- **开发环境**： Keil MDK/VS CODE
- **依赖库**：STM32 HAL 库, FreeRTOS
- **调试工具**：串口调试助手 (波特率 115200)
